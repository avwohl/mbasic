    Execution Flow¶

    1. Parse: Lexer → Parser → AST
    2. Setup: Create WebIOHandler, Runtime, Interpreter
    3. Breakpoints: Set interpreter.breakpoints
    4. Execute: execute_ticks() with ui.timer()
    5. Tick Loop:
    6. Call interpreter.tick()
    7. Update variables/stack windows
    8. Check for 'BREAK', 'DONE', or continue
    9. Schedule next tick with ui.timer(0.01, ...)

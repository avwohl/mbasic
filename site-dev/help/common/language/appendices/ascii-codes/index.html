    Usage in BASIC-80¶

    Converting Between ASCII and Characters¶

    10 REM ASCII to character
20 C$ = CHR$(65)        ' Returns "A"
30 PRINT C$

40 REM Character to ASCII
50 N = ASC("A")         ' Returns 65
60 PRINT N

    Common ASCII Values¶

    • Letters: A-Z = 65-90, a-z = 97-122
    • Digits: 0-9 = 48-57
    • Space: 32
    • Newline: CR=13, LF=10
    • Tab: 9

    String Functions Using ASCII¶

    • ASC - Get ASCII code of first character
    • CHR$ - Get character from ASCII code
    • INKEY$ - Read keyboard (returns ASCII)
    • INPUT$ - Read characters

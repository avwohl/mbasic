## Help Navigation

Use these keys while viewing help:

| Key | Action |
|-----|--------|
| **↑/↓** | Scroll through current page |
| **Tab** | Move to next link |
| **Enter** | Follow the highlighted link |
| **U** | Go back to previous page |
| **ESC** or **Q** | Close help and return to editor |

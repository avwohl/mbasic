# OUT

## Syntax

```basic
OUT I,J
where I and J are    integer   expressions     in   the
range 0 to 255.
```

## Purpose

To send a byte to a machine output port.

## Remarks

The integer expression I is the port number, and the integer expression J is the data to be transmitted.

## Example

```basic
100 OUT 32,100
```

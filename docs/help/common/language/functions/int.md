# INT

## Syntax

```basic
INT (X) Versions,:     8K, Extended, Disk
```

## Description

Returns the largest integer <=X. Examples:      PRINT INT ( 99. 89) 99 Ok PRINT INT(-12.ll} -13 Ok See the FIX and CINT functions which also return integer values.
